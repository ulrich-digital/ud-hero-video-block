# ud-hero-video-block

Ein Block für einen Hero-Bereich mit Hintergrundvideo, Posterbild und Overlay-Text.

## Funktionen

- Hintergrundvideo mit autoplay, muted, loop und playsinline
- Eyebrow und Headline direkt im Editor editierbar
- Video und Posterbild per Medienauswahl oder URL
- Optionales Ausblenden des Videos auf mobilen Geräten
- Serverseitiges Rendering per render_callback
